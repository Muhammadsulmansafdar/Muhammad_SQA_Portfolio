The Project Context

I was responsible for testing how Canva handles design changes when a user is completely offline or has a very poor internet connection.

This was a high-responsibility task because if the offline logic fails, the user loses their work. My job was to make sure that everything—from simple text edits to heavy audio and video recordings—stays safe on the device and syncs correctly to the account the moment the internet comes back on.

Cross-Platform Sync Testing

I tested the transition between online and offline states repeatedly. I checked if the "Available Offline" toggle actually worked and ensured the mandatory reload prompts triggered correctly so the app could cache the design data properly.

Editor Stability & Media

I focused on the editor's behavior while disconnected. This included testing if audio elements still played, if shapes and colors saved correctly, and making sure the app didn't crash or show "Something went wrong" errors during sync.
